## 📌 Commit 메시지 규칙
학습 진행 상황을 효과적으로 추적하기 위해 다음과 같은 commit 메시지 규칙을 따릅니다:

```
[섹션번호] 주제: 상세 내용
- 학습한 주요 개념
- 실습한 내용
- 특이사항 또는 주의점
```

예시:
```
[섹션4] 객체 지향 원리 적용: 새로운 할인 정책 개발 완료
- 새로운 정률 할인 정책 구현
- OCP, DIP 원칙 적용 시도
- 추상화에 의존하도록 설계 변경
```
